# How to run this project

### Clone the repository
    git clone https://github.com/zacharynoble/react-typescript-vite-tailwind-auth-template.git
    cd react-typescript-vite-tailwind-auth-template


### Install Packages
    npm install


### Start the application
    npm run dev


### Visit the application in your browser at:
    http://localhost:3000


### Sample .env file
    VITE_NODE_ENV = DEVELOPMENT
    VITE_API_URL = http://127.0.0.1:8080/api
